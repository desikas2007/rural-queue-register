## 🚀 QUICK START

### 1. Install Dependencies
```bash
npm install
```

### 2. Configure Environment
```bash
cp .env.example .env.local
# Edit .env.local with your credentials
```

### 3. Setup Database
```bash
npm run prisma:migrate
npm run seed  # Optional: add demo data
```

### 4. Start Development
```bash
npm run dev
```

### 5. Access Application
```
http://localhost:3000
```
